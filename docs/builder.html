      物種資訊

      ✅ 打包完成！接著把它變成你的網頁

      下載的 my-pca-dashboard.zip 就是一個完整的無伺服器網站。三步驟上線：

      1. 建立 GitHub repo：到 GitHub 按 New repository，取個名字（例如 my-pca），建立。
      2. 上傳檔案：解壓下載的 zip，把裡面所有檔案與資料夾（index.html、css/、js/、vendor/、data/、.nojekyll）拖到 repo 的「Add file → Upload files」，Commit。
      3. 啟用 Pages：repo → Settings → Pages → Source 選 Deploy from a branch → Branch 選 main、Folder 選 /(root) → Save。約 1–2 分鐘後，你的互動式 PCA 網頁就在 https://<帳號>.github.io/<repo>/ 上線了。

      整個網站純靜態、無伺服器、可離線開啟；資料已內嵌，觀看者不需再上傳。日後要換資料，回到本精靈重新打包即可。
